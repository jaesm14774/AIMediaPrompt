# notion

極簡 SaaS 儀表板美學，生產力工具風格

## 色彩

- 主色：近黑 (#1F1F1F)、Notion 藍 (#2383E2)
- 背景：淺灰 (#F7F7F5)、純白 (#FFFFFF)
- 強調：成功綠 (#0F7B6C)、警示紅 (#E03E3E)、警告黃 (#DFAB01)

## 視覺元素

- 卡片式佈局，細微邊框
- 乾淨的數據視覺化
- 標籤和標記元素
- 圖標式導航提示
- 進度條和指標顯示
- 簡潔的手繪線條插圖

## 字體風格

- System UI 或 Inter 風格，乾淨實用

## 最適合

產品展示、SaaS 簡報、生產力內容、B2B 主題
